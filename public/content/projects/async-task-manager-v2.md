## System Architecture
A high-performance asynchronous system designed for large-scale data processing.

### Capabilities
* **Distributed Scraping**: Built a Django REST API to scrape internet data, serving JSON responses to 1,000+ active users.
* **AI Summarization**: Automated data summarization using GenAI.
* **Intelligent Interface**: Implemented a chatbot interface leveraging MCP (Model Context Protocol) servers for deeper insights.
