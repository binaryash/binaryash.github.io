
# BinaryAsh Portfolio website  

The site is built using Jekyll, a static site generator, to provide a fast, responsive, and customizable platform for showcasing my projects and skills. This is a fast and minimalistic blog theme with a clean dark mode. It offers features like light & dark mode with a theme switcher, various list styles, landing page with navbar and footer, and more.
