## Streamlined Invoicing
A utility-focused application built for speed and professional output. It simplifies the administrative burden of freelance and small business billing.

### Technical Highlights
* **React & Ant Design**: Built with React and the Ant Design component library for a polished, professional UI that prioritizes user efficiency.
* **Client & Catalog Management**: Includes a persistent database for client details and reusable item catalogs, reducing repetitive data entry.
* **On-the-Fly PDF Generation**: Integrated PDF export capabilities, allowing users to generate and download professional invoices directly from the browser.
* **Modern Build Workflow**: Utilizes **Vite** for an extremely fast development experience and optimized production builds, ensuring the app loads instantly.
